# tpmp-lab2-task4

## Overview
Проект для лабораторной работы №2, задание 4.  
Программа на языке C для обработки текстовых файлов: чтение, обработка данных согласно варианту и запись результатов в новый файл.

## Usage
1. Клонировать репозиторий:
   ```bash
   git clone git@github.com:NgdMav/tpmp-lab2-task4.git
   cd tpmp-lab2-task4
   ```
2. Собрать программу:
   ```bash
   make
   ```
3. Подготовить входной файл `data/input.txt`
4. Запустить:
   ```bash
   ./bin/file_processor
   ```
5. Результат будет в `data/output.txt`
6. Очистка:
   ```bash
   make clean
   ```

## Additional Notes
- Программа реализована по модели КИС-2.
- Сборка автоматизирована через Makefile.
- Настроен GitHub Actions для автоматической сборки.
- Входные данные читаются из `data/input.txt`
- Результаты сохраняются в `data/output.txt`
