## 6. Выводы
Программа успешно:
- Читает данные из текстового файла
- Парсит структурированные данные в формате CSV с разделителем ";"
- Фильтрует страны по заданным критериям (материк Евразия, население < 20 млн)
- Выводит отфильтрованные данные на экран в табличном формате
- Сохраняет результаты в новый файл
- Все тесты пройдены успешно
